# BOM Engine Preview

A desktop application for managing your video production pipeline.

---

## Dashboard

The main dashboard shows your key metrics at a glance:
- Videos created this month
- Pending reviews
- Active clients
- API spend tracking

## Settings

### API Keys
Configure your video generation services:
- Replicate API Token
- ElevenLabs API Key
- Anthropic API Key
- HeyGen API Key (Optional)
- Google Drive integration

![Settings - API Keys](screenshots/Screenshot%202025-12-02%20at%2010.29.02.png)

### Default Voice
Choose from multiple voice options:
- Dutch Male
- Dutch Female
- English Male
- English Female

![Settings - Voice](screenshots/Screenshot%202025-12-02%20at%2010.29.28.png)

### Default Video Format
Select your preferred output format:
- Vertical (9:16) — for TikTok, Reels, Shorts
- Square (1:1) — for Instagram feed
- Horizontal (16:9) — for YouTube, LinkedIn

![Settings - Format](screenshots/Screenshot%202025-12-02%20at%2010.29.45.png)
